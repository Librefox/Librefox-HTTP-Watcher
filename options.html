Firefox Themes :
Firefox integreted themes are hidden from webextensions (bug 1386004/1414512).
If you are using one of the 3 defaults integrated themes, please select the used one.
This setting purpose is not to change your theme but to let the extension know what theme is used.
Most of other themes are handled automatically but because of (bug 1386004/1414512) themes
using old api are not supported, and theme downloading images are not supported too.
This setting is used to restore the used theme after displaying the red bar for an http url.
Also this settings is ignored when using a compatible theme.






Red Color Warning :
Turn the whole browser red on http sites or just the url bar.
For unsupported themes unexpected color behaviour may occur
because used theme colors are not be accessible (bug 1386004/1414512).




Supported Themes :
- Default firefox integrated themes through settings above
- All themes using new theming system (like Ark-Dark/Ark-Theme)
- Firefox Color (Firefox Test Pilot)


Supported Extensions :
All theming extensions using new theming system like :
- Firefox Color (Firefox Test Pilot)
- QuantumEdge Dark
- Breeze Dark
- etc.

UnSupported Themes :
- Connected themes (that download images and backgrounds)
- Themes set throught "lightweightThemes.usedThemes"
- Themes using old theming system
